Overview

Hello and welcome to Mock Market. This is a simple, front-end application designed to provide concise, data-backed stock summaries for S&P 500 companies. It combines a clean user interface with real-time data and a direct, no-nonsense narrative style to help you grasp performance trends quickly.

Usage

  1. Find Your Stocks: Start typing an S&P 500 company name or ticker into the search bar. We’ll display matches if they exist.
  2. Add up to 3 Tickers: Click the plus button to add them to your list. If you try for a fourth, we’ll remind you to keep it streamlined.
  3. Generate the Report: Hit “Get Report” to fetch real historical data, then watch as OpenAI crafts a direct, no-holds-barred summary.
  4. Restart Any Time: Checked out your stocks and want a fresh start? Click “Restart” and pick new tickers.

Local JSON for S&P 500

To keep performance snappy, I rely on a local JSON file for all S&P 500 listings. This prevents extra API calls just to autocomplete your search. It’s fast, straightforward, and ensures we don’t hit external service limits when you’re simply browsing tickers.

Vercel Functions

This project is deployed on Vercel, using their serverless functions to handle key tasks. One function pulls real-time data from Polygon.io, while another sends your chosen stocks to OpenAI. No dealing with servers or complicated infrastructure—it scales nicely as the app grows.

Polygon & OpenAI

For data, I tap into Polygon.io to retrieve historical pricing, volume, and performance metrics. Each request is packaged into a tidy object, then forwarded to OpenAI with a carefully crafted prompt. That prompt instructs the model to be informative, concise, and even a bit blunt—giving you straightforward advice in a user-friendly format.

Background Animation

If you’ve seen the gentle, drifting lines on the main page, that’s a modified snippet from a fantastic CodePen project by @tati420. I adjusted the spacing, color, and drifting behavior to fit the app’s style while retaining that tranquil, techy vibe.
